> From 2020-10-05 to 2021-09-06

Apprenticeship at NextSourcia working on AquilaCMS.

AquilaCMS is a product maintained by NextSourcia

AquilaCMS Development (e-commerce CMS):

- An open-source e-commerce web application
- A 100% JavaScript solution with MERN Stack
- A multi-themes website: Front-end can be used with any JS technology (ReactJS, VueJS, Angular, etc)
- A complete back office to manage everything in your website

Worked with:

- AngularJS
- NodeJS
- ReactJS
- NextJS
- MongoDB
- Cypress (combined with bash scripts for daily automatic front-end tests)
- Mocha (API test)
- GIT and github
- Trello
- Creation of documentation for plugin integration in Aquila
- Creation of some Aquila plugins
