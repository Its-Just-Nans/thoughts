> Tutored Project with Thales Alenia Space

The project (made in group of 6) was to update a moving robot and adapt a spacial Camera (ToF - Time Of Flight) to it.

The robot was used to demonstrate some sattelite trajectories.

Worked with :

- Raspberry Pi
- API of a spatial camera (Basler ToF Camera)
- Fusion 360
- FabLab

Links:

- https://youtu.be/Xz0L6iHNSwA - presentation of the project
- https://github.com/Its-Just-Nans/thales-project

From 2020-04 to 2021-06
